# AI-Powered Dry Fruit E-Commerce Website

A production-grade full-stack e-commerce web application for selling dry fruits online, built using React, Node.js, Express, and MySQL.

This project includes modern e-commerce features such as:

## Features

- User Authentication (Customer + Admin Login)
- Product Catalog with Categories
- Product Search and Filtering
- Shopping Cart with Stock Validation
- Checkout and Razorpay Test Payment Integration
- Order Management
- Order Cancellation with Inventory Restock
- Real-Time Inventory Management
- Wishlist
- Product Reviews & Ratings
- AI-Powered Nutrition Assistant
- Admin Dashboard
- Low Stock Alerts
- Dynamic Stock Tracking
- Responsive Modern UI

## Tech Stack

Frontend:
- React

Backend:
- Node.js
- Express

Database:
- MySQL

Payment:
- Razorpay (Test Mode)

## Highlights

- Full relational database design
- Inventory auto deduction after successful orders
- Inventory restoration on order cancellation
- Product-level AI assistant for nutrition guidance
- Production-style e-commerce architecture
